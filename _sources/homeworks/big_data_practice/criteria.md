# 评分标准

60分是学校的及格线，80分是量潮认可这门课的教学效果的及格线。
也就是说，这门课如果在我们的评分标准下达到80分，我们就认为这门课我们的教学效果已经达到了，作为我们的实习生已经能力合格；否则就还需要经过实训并合格才能达到我们的实习标准。

设计报告的80分起标准最简版：架构设计图是关键。前面已经提到，生产用的架构设计文档少不了通过关键示意图表达想法，就像盖房子要画图纸一样，架构设计文档就是代码的图纸，代码就是砖头。

最终报告的80分起标准最简版：能够跑出来正确的CPI图，包括趋势、图例都是正确的。趋势正确但是图例不正确的同学70分起。具体的标准我们还在细化中。
